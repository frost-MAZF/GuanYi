使用Visual Studio

使用Visual Studio相对简单，可以直接下载微软的Visual Studio Community,就可以直接使用了。

Visual Studio Community的安装请选择供学生,开放源代码和个人免费使用的那个,如下图所示.,选择第三个

C语言编译环境

C语言的勾选和C++差不多,就不说了.

C++编译环境配置

在Visual Studio Community中勾选,当然你可以根据自己需求去勾选.




Node.js

Node.js请勾选这个

HTML

HTML可以直接拿浏览器看,就不说了.

Python

Python请勾选这个

至于别的东西就自己勾选就可以了。

测试

接下来我们进行一下测试,就拿C语言好了.

先点击创建新项目


然后选择C++

然后点击空项目

随后的这些选择一个喜欢的就行

然后右边点击源文件,添加,新建项.

当我们输入完代码后点击上方的本地windows调试器

随后在出现的命令提示符里,就是你的代码输出了

像Linux一样使用终端

相比较直接使用Visual Studio来说，直接使用终端编译运行不会那么臃肿，也不用等待Visual Studio那漫长的启动时间，你只需要一个文本编辑器，以及一个编译器就可以了。
